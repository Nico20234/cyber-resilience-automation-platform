# Automated Cyber Resilience Platform

Proyecto de automatización orientado a resiliencia operativa utilizando Python, Docker, Ansible y Terraform.

## Objetivo

Simular un entorno productivo donde un servicio pueda ser monitoreado y recuperado automáticamente ante fallos.

## Tecnologías

- Python
- Flask
- Docker
- Ansible
- Terraform
- Linux
- Git

## Arquitectura

Servicio Flask + monitoreo automático + recuperación mediante infraestructura como código.
